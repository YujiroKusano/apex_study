## 数式で使う正規表現
Regix関数・・・正規表現の値と一致する場合はTRUEが変える
第一引数・・・判定対象文字列
第二引数・・・正規表現文字列

正規表現例

### 範囲

例) 0~9までの文字列
```“[0-9]”```

例) 大文字のアルファベットのいずれかの文字列
```“[A-Z]”```

例) 小文字のアルファベットのいずれかの文字列
```“[a-z]"```

例) カタカナのいずれかの文字列
```“[ァ-ヴ]”```

例) 組み合わせ
```“[0-5A-Ea-c]"```


例) abcのいずれかの文字列
```“[abc]”```

例) abcのいずれでもない文字列
```“[^abc]”```


### 文字数

例) 3文字の文字列
```“{3}”```

例) 任意の０文字以上
```“.*"```
