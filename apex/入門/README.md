## プログラムの常識
- プログラムは通常上から１行ずつ順番に処理が行われていく。
- 処理はブロックと呼ばれる```{```と```}```の間で行われる。
- ブロックごとに処理が分かれている。
- 処理を行わない文字を```コメント```と呼ぶ
    - コメントの記載方法
    ```apex
    // 単数行コメント １行分のコメントをかけます
    /** 
    コメント 
    複数行かけます
    */
    ```
