## プログラムの常識
- プログラムは通常上から１行ずつ順番に処理が行われていく。
- 処理はブロックと呼ばれる```{```と```}```の間で行われる。
- ブロックごとに処理が分かれている。
- 呼ばれるプログラム処理を行わない文字を```コメント```と呼ぶ
    - コメントの記載方法
        - 単数行コメント・・・```// コメント```
        - 複数行コメント・・・```/** コメント */```
